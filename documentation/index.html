HTML

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. Please visit http://getbootstrap.com/ to learn more about Bookstrap. For a detailed information about its grid system, click on http://getbootstrap.com/css/#grid.

See how aspects of the Bootstrap grid system work across multiple devices with a handy table.

Extra small devices Phones (<768px) Small devices Tablets (≥768px) Medium devices Desktops (≥992px) Large devices Desktops (≥1200px)
Grid behavior Horizontal at all times Collapsed to start, horizontal above breakpoints
Container width None (auto) 750px 970px 1170px
Class prefix .col-xs- .col-sm- .col-md- .col-lg-

Simple grid layout looks like following:

<div class="row">
    <div class="col-sms-6 col-sm-6">
    </div>
    <div class="col-sms-6 col-sm-6">
    </div>
</div

 

Remember that col-sm- only works for Small Devices(≥768 and < 992).

Customizing menu

The menu contain six top-level menu items, each item has submenus. The structure of a top-level menu with multiple submenus is as below.


    <li class="dropdown">
        <a href="#" class="dropdown-toggle" data-toggle="dropdown">RSVP<b class="caret"></b></a>
        <ul class="dropdown-menu">
            <li><a href="rsvp-1.html">rsvp 1</a></li>
            <li><a href="rsvp-1.html">rsvp 2</a></li>
        </ul>                          
    </li>

To remove the submenus from the above top-level menu, replace the previous code by the following.

    <li>
        <a href="rsvp-1.html">RSVP</a>
    </li>

Countdown

You can change the event date in countdown section located in js/script.js. The date format is Year/Month/Day.

In the below example the date is 16 December 2016, so the date format is 2016/12/16.

$('#clock').countdown('2016/12/16').on('update.countdown', function (event) {
    var $this = $(this).html(event.strftime(''
            + '<span class="wraper-clock"><div class="value">%D</div><div class="value-label">Days</div></span><span class="sep">:</span>'
            + '<span class="wraper-clock"><div class="value">%H</div><div class="value-label">Hours</div></span><span class="sep">:</span>'
            + '<span class="wraper-clock"><div class="value">%M</div><div class="value-label">Minutes</div></span><span class="sep">:</span>'
            + '<span class="wraper-clock"><div class="value">%S</div><div class="value-label">Seconds</div></span>'
            ));
});

PHP Form

Open the PHP file include/contact.php and replace youremail@domain.com by your email.

$to = 'youremail@domain.com';
$headers = 'From: youremail@domain.com' . "\r\n";

To prevent uers from submeting the form with an empty input field value, we have to add the rule required to each input form field that we consider mandatory. If you want to add new input fields or specify your own error messages during the validation call, just open js/script.js and edit the section Ajax Form Submit. 

                  
$("#block-main-form").validate({
    rules: {
        name: "required",
        subject: "required",
        message: "required",
        email: {
            required: true,
            email: true
        }
    },
    messages: {
        name: "Name Required",
        subject: "Subject Required",
        message: "Message Required",
        email: {
            required: "Email Address Required",
            email: "Please enter a valid email address"
        }
    },

Please note that we are also validating all input fields via PHP, see PHP file include/contact.php.

Now we are ready to submit the contact form by performing an Ajax Call to the contact PHP file include/contact.php.

Loading time is set in this case to 4000ms.

                  
$("#block-main-form").validate({
    submitHandler: function (form) {
        $.ajax({
            type: "POST",
            url: "include/contact.php",
            data: $(form).serialize(),
            success: function (response) {
                $('#success').html(response);
                var $this = $('.btn');
                $this.button('loading');
                setTimeout(function () {
                    $this.button('reset');
                }, 4000);
            }
        });
        return false;
    }
});

In this theme we are using a very simple HTML5 email template, it is located in include/email-template.html.

// load HTML email template
$body = file_get_contents('../include/email-template.html');

  • Creating an empty php file for example form-testing.php.
  • Add the code below to form-testing.php and execute it in your browser. It should display some text as a result if it works. 
<?php

    echo file_get_contents('../include/email-template.html');

?>

If it is still not working, you can try to replace '../include/email-template.html' by '/../include/email-template.html' or by './include/email-template.html'
